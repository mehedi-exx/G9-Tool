G9Tool কী?

G9Tool একটি শক্তিশালী এবং ব্যবহারকারী-বান্ধব প্রোডাক্ট জেনারেটর টুল যা আপনাকে সহজেই আপনার ই-কমার্স প্রোডাক্টের জন্য আকর্ষণীয় HTML কোড তৈরি করতে সাহায্য করে।

HTML কোড জেনারেশন

স্বয়ংক্রিয়ভাবে প্রোডাক্টের জন্য সুন্দর HTML কোড তৈরি করুন

ডেটা সংরক্ষণ

আপনার প্রোডাক্ট তথ্য সংরক্ষণ এবং পরিচালনা করুন

রেসপনসিভ ডিজাইন

বহুভাষিক সাপোর্ট

বাংলা এবং ইংরেজি ভাষায় ব্যবহার করুন

টিপ: G9Tool সম্পূর্ণ ফ্রি এবং কোনো রেজিস্ট্রেশনের প্রয়োজন নেই। শুধু লগইন করে ব্যবহার শুরু করুন!

কীভাবে শুরু করবেন

1

লগইন করুন

প্রথমে index.html পেজে গিয়ে আপনার ইউজারনেম এবং পাসওয়ার্ড দিয়ে লগইন করুন। ডিফল্ট ইউজার: admin, পাসওয়ার্ড: admin

2

ড্যাশবোর্ডে প্রবেশ

সফল লগইনের পর আপনি স্বয়ংক্রিয়ভাবে ড্যাশবোর্ডে চলে যাবেন যেখানে সব টুলস পাবেন

3

ভাষা নির্বাচন

সাইডবার মেনু থেকে আপনার পছন্দের ভাষা (বাংলা/ইংরেজি) নির্বাচন করুন

4

ফিল্ড কাস্টমাইজ (ঐচ্ছিক)

প্রয়োজন অনুযায়ী ফর্ম ফিল্ডগুলো চালু/বন্ধ করুন Field Customize সেকশন থেকে

5

প্রোডাক্ট তৈরি শুরু করুন

এখন আপনি প্রোডাক্টের তথ্য দিয়ে HTML কোড জেনারেট করতে পারেন

গুরুত্বপূর্ণ: প্রথমবার ব্যবহারের আগে অবশ্যই Field Customize সেকশনে গিয়ে আপনার প্রয়োজনীয় ফিল্ডগুলো সেট করে নিন।

বাধ্যতামূলক ফিল্ড

ঐচ্ছিক ফিল্ড

জেনারেট প্রক্রিয়া

1

তথ্য পূরণ করুন

সব প্রয়োজনীয় ফিল্ড পূরণ করুন

2

"জেনারেট" বাটনে ক্লিক

সব তথ্য দেওয়ার পর জেনারেট বাটনে ক্লিক করুন

3

প্রিভিউ দেখুন

আউটপুট সেকশনে আপনার প্রোডাক্টের প্রিভিউ দেখুন

4

কোড কপি করুন

"কপি করুন" বাটনে ক্লিক করে HTML কোড কপি করুন

প্রো টিপ: একাধিক ছবি যোগ করতে "আরও ছবি যোগ করুন" বাটন ব্যবহার করুন। কাস্টম তথ্যের জন্য "কাস্টম তথ্য যোগ করুন" ব্যবহার করুন।

ফিল্ড কাস্টমাইজেশন

ফিল্ড কাস্টমাইজেশন ফিচার আপনাকে ড্যাশবোর্ডে প্রদর্শিত ফর্ম ফিল্ডগুলো নিয়ন্ত্রণ করার সুবিধা দেয়। এর মাধ্যমে আপনি শুধুমাত্র প্রয়োজনীয় ফিল্ডগুলো দেখাতে পারেন।

কীভাবে ব্যবহার করবেন

1

Field Customize এ যান

সাইডবার মেনু থেকে "⚙️ ফিল্ড কাস্টমাইজ" এ ক্লিক করুন

2

ফিল্ড নির্বাচন করুন

যে ফিল্ডগুলো দরকার সেগুলো চেক করুন, অপ্রয়োজনীয়গুলো আনচেক করুন

3

প্রিভিউ দেখুন

নিচের প্রিভিউ সেকশনে সক্রিয় ফিল্ডগুলো দেখুন

4

আপডেট করুন

"আপডেট করুন" বাটনে ক্লিক করে সেটিংস সেভ করুন

ফিল্ড প্রকারভেদ

বাধ্যতামূলক ফিল্ড

এগুলো সবসময় দেখানো হবে এবং বন্ধ করা যাবে না

ঐচ্ছিক ফিল্ড

এগুলো আপনার প্রয়োজন অনুযায়ী চালু/বন্ধ করতে পারেন

মনে রাখুন: ফিল্ড সেটিংস পরিবর্তনের পর ড্যাশবোর্ডে ফিরে গিয়ে পরিবর্তনগুলো দেখুন।

অ্যাডমিন প্যানেল

অ্যাডমিন প্যানেল আপনার সংরক্ষিত প্রোডাক্টগুলো পরিচালনার জন্য একটি শক্তিশালী টুল। এখানে আপনি সব প্রোডাক্ট দেখতে, এডিট করতে, ডিলিট করতে এবং ভেরিফাই করতে পারেন।

WhatsApp অর্ডার সিস্টেম

G9Tool এর একটি বিশেষ ফিচার হলো স্বয়ংক্রিয় WhatsApp অর্ডার সিস্টেম। যখন কাস্টমার "Order Now" বাটনে ক্লিক করবে, তখন স্বয়ংক্রিয়ভাবে একটি অর্ডার মেসেজ তৈরি হয়ে WhatsApp এ পাঠানো হবে।

ভিডিও ইন্টিগ্রেশন

YouTube ভিডিও লিংক দিলে স্বয়ংক্রিয়ভাবে প্রোডাক্ট পেজে ভিডিও এম্বেড হয়ে যাবে। এটি কাস্টমারদের প্রোডাক্ট সম্পর্কে আরও ভালো ধারণা দিতে সাহায্য করে।

কাস্টম স্টাইলিং

জেনারেট করা HTML কোডে CSS স্টাইল ইনক্লুড থাকে যা আপনি প্রয়োজন অনুযায়ী কাস্টমাইজ করতে পারেন:

/* প্রোডাক্ট কার্ডের রঙ পরিবর্তন */ .product-card { background: #your-color; border: 2px solid #your-border-color; } /* বাটনের স্টাইল */ .order-btn { background: linear-gradient(45deg, #color1, #color2); }

থিম ডাউনলোড

সাইডবার মেনুতে "G9Theme" বক্স থেকে আপনি ব্লগার থিম ডাউনলোড করতে পারেন। এই থিমটি G9Tool এর সাথে সামঞ্জস্যপূর্ণ এবং আপনার ব্লগে প্রোডাক্ট প্রদর্শনের জন্য অপ্টিমাইজড।

সতর্কতা: কাস্টম CSS যোগ করার সময় সাবধান থাকুন যাতে রেসপনসিভ ডিজাইন নষ্ট না হয়।
প্রো টিপ: আরও ভালো ফলাফলের জন্য উচ্চ মানের প্রোডাক্ট ছবি ব্যবহার করুন এবং বিস্তারিত বর্ণনা দিন।
