১. G9Tool কী?

G9Tool একটি HTML-ভিত্তিক প্রোডাক্ট জেনারেটর টুল যা আপনাকে আপনার প্রোডাক্টের কোড তৈরি, প্রিভিউ, কপি, সংরক্ষণ, ওয়াটসঅ্যাপ অর্ডার বাটন, এবং ইউটিউব ভিডিও এম্বেড করার সুবিধা দেয়।

২. কীভাবে প্রোডাক্ট কোড তৈরি করবেন?

Dashboard এ গিয়ে প্রোডাক্ট নাম, কোড, ছবি, মূল্য ইত্যাদি তথ্য দিন এবং "Generate" বাটনে ক্লিক করুন। এরপর "Copy" বাটনে ক্লিক করে কোড কপি করে নিন।

৩. Field Manager এর কাজ কী?

আপনি যেসব ইনপুট ফিল্ড ব্যবহার করতে চান বা লুকিয়ে রাখতে চান, তা Field Manager থেকে নিয়ন্ত্রণ করতে পারবেন। এটি আপনার ফর্ম কাস্টমাইজ করার সহজ উপায়।

৪. Admin Panel

এখানে আপনি আপনার সংরক্ষিত প্রোডাক্টগুলো দেখতে পাবেন, যেকোনো প্রোডাক্ট Edit / Delete / Preview করতে পারবেন। Export / Import সুবিধাও আছে।

৫. কোড কপি ও প্রিভিউ

Output অংশে আপনি শুধু প্রিভিউ দেখতে পাবেন। কোড কপি করার জন্য Copy বাটন চাপলেই কোড কপি হয়ে যাবে — কোন কোড দেখাবে না।

৬. WhatsApp অর্ডার

প্রোডাক্টের নিচে “Order Now” বাটনে ক্লিক করলে স্বয়ংক্রিয়ভাবে WhatsApp-এ অর্ডার মেসেজ তৈরি হয়ে যাবে।
